## ⚙️ 發布前必須設置

### 🚨 重要：修改 5 處代碼

#### 1. PayPal 用戶名（2處）

**文件 1：`payment.html`（第 185 行）**
```html
當前：value="https://paypal.me/YOURUSERNAME"
改為：value="https://paypal.me/您的PayPal用戶名"
```

**文件 2：`js/payment.js`（第 92 行）**
```javascript
當前：const paypalLink = `https://paypal.me/YOURUSERNAME/...
改為：const paypalLink = `https://paypal.me/您的PayPal用戶名/...
```

#### 2. Wise 郵箱（3處）

**文件 1：`payment.html`（第 250 行）**
```html
當前：value="your-wise-email@example.com"
改為：value="您的Wise郵箱@example.com"
```

**文件 2：`payment.html`（第 295 行）**
```html
當前：<strong>your-wise-email@example.com</strong>
改為：<strong>您的Wise郵箱@example.com</strong>
```

**文件 3：`js/payment.js`（第 96 行）**
```javascript
當前：const wiseEmail = 'your-wise-email@example.com';
改為：const wiseEmail = '您的Wise郵箱@example.com';
```
