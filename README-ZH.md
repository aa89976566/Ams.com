### 1. 本地測試
```bash

open index-zh.html

# 或使用本地伺服器（推薦）
python -m http.server 8000
# 然後訪問 http://localhost:8000/index-zh.html
```

### 2. 部署到伺服器
```bash
# 上傳所有文件到伺服器
# 保持文件夾結構不變
scp -r * user@server:/path/to/website/

# 確保文件權限正確
chmod 644 index-zh.html css/*.css js/*.js
chmod 755 images/
```

### 3. 使用 GitHub Pages
```bash
# 1. 創建 GitHub 倉庫
# 2. 上傳所有文件
git init
git add .
git commit -m "Initial commit - Patrick Writer Chinese Version"
git remote add origin https://github.com/username/patrick-writer-zh.git
git push -u origin main

# 3. 在 GitHub Settings > Pages 中啟用 GitHub Pages
# 4. 選擇 main 分支
# 5. 網站將在 https://username.github.io/patrick-writer-zh/index-zh.html 上線
```
